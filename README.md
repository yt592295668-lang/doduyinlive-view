# Project Documentation for doduyinlive-view

## Overview
This repository contains the source code for the doduyinlive-view project, which aims to provide a seamless live-viewing experience. 

## Features
- Real-time live streaming capabilities
- User-friendly interface
- Support for multiple video formats
- Chat feature during live sessions
- Archive of past broadcasts

## Installation
To install the doduyinlive-view project, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/yt592295668-lang/doduyinlive-view.git
   ```
2. Change to the project directory:
   ```bash
   cd doduyinlive-view
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm start
   ```

## Usage
After installing the project, you can start live streaming by navigating to `http://localhost:3000` in your web browser. Follow the on-screen instructions to set up your live stream.

## Technical Stack
- **Frontend:** React.js
- **Backend:** Node.js with Express
- **Database:** MongoDB
- **Streaming:** WebRTC
